## 🧭 What codex-switcher does

codex-switcher is a macOS menu bar app that helps you manage more than one OpenAI Codex account. When one account hits a usage limit, the app switches to the next account for you.

That means you do not need to log out and log in by hand each time a limit is reached. The app keeps the process simple and stays in the background while you work.
